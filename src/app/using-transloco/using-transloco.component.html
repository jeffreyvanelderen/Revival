using-transloco works!

Using ng-container *transloco="let t"

{{Language.NL}}

{{t('helloX', {x: 'Jeffrey'})}}

Using ng-container *transloco="let t; lang: Language.FR"

{{Language.FR}}

{{t('helloX', {x: 'Jeffrey'})}}

Using ng-container *transloco="let t; prefix: 'nested'"

{{t('nestedProperty')}}

Using pipe

{{'helloX' | transloco : {x: 'Jeffrey'} }}

Using attribute directives
